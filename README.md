## 👋 About This Portfolio

This repository documents my hands-on journey learning **Microsoft Power BI** as part of my Master's in Data Science program. Each project folder contains:

- 📁 `.pbix` — Power BI report file
- 📄 `README.md` — Project overview, key insights, and screenshots
- 📂 `data/` — Source datasets used (where publicly shareable)
- 🖼️ `screenshots/` — Dashboard previews

> **Note:** This is a living portfolio — new projects and study reports are added regularly as I explore new Power BI features and real-world datasets.
