### Layoff Dashboard
> *An interactive Power BI dashboard analyzing global layoff trends across tech and non-tech industries.
Explores which companies, countries, and sectors were hit hardest — and when. Built using a real-world
dataset with 2,000+ records spanning 2020–2026, featuring dynamic slicers, KPI cards, geo maps,
and trend analysis to uncover the human story behind the numbers.*

**| Detail | Info |**
| 📅 Date | 2020-2026 |
| 🏷️ Category | HR Analytics · Labor Market |
| 🛠️ Tools | Power BI · DAX · Power Query |
| 📊 Dataset | Kaggle - International layoff records across companies & industries |

**Key Features:**
- 📉 Trend analysis of layoffs across years and countries
- 🗺️ Geographic breakdown using filled map visual
- 🏢 Company-wise and sector-wise comparison
- 📌 KPI cards for total layoffs, affected companies, and top impacted sectors
- 🔍 Interactive slicers — filter by year, country, industry, and company size

**Skills Demonstrated:** `DAX Measures` · `Filled Map` · `Conditional Formatting` · `Drill-through` · `Bookmarks`
